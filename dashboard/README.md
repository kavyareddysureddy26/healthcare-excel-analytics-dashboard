# Excel Dashboard & Visual Outputs

## Dashboard Overview

This folder contains the **interactive Excel dashboard** created as part of the **Healthcare Claims & Cost Analysis** project.

The dashboard is designed for **non-technical stakeholders** (management, operations, finance teams) to quickly understand healthcare utilization, payer coverage, and cost risk using KPIs and visual analytics.

## Files Included
- **PROJECT.xlsx**  
Contains the complete Healthcare Claims Dashboard, including:
  - Cleaned and transformed data
  - Pivot tables used for analysis
  - KPI calculations
  - Interactive dashboard with charts and slicers

- **Dashboard_Screenshot1.png**  
- **Dashboard_Screenshot2.png**  
  Preview images of the dashboard for quick viewing without opening Excel.

## Dashboard Features
- **Executive-level KPI summary**, including:
  - Total Unique Patients  
  - Total Number of Encounters  
  - Total Healthcare Cost  
  - Total Payer Coverage Amount  
  - Total Uncovered Claim Amount  

- **Interactive charts**, such as:
  - Encounter type distribution (ambulatory, inpatient, outpatient, emergency, etc.)
  - Top payers by total coverage amount
  - City-wise healthcare claim costs
  - Year-over-year encounter trends

- **Slicers for interactivity**, allowing users to:
  - Filter by patient city
  - Dynamically update KPIs and charts
  - Explore patterns across locations and time

## Purpose of the Dashboard
The dashboard helps answer key business questions such as:
- Which encounter types generate the highest volume?
- Which payers contribute the most to healthcare coverage?
- How much cost remains uncovered and poses financial risk?
- Which cities account for the highest healthcare spending?
- How have encounter volumes changed over time?

## Usage Instructions
1. Download **PROJECT.xlsx** from this folder  
2. Open the file in Microsoft Excel (desktop version recommended)
3. Navigate to the **Dashboard** sheet
4. Use slicers to filter by **City** and explore insights
5. Review KPIs and charts for decision-ready insights

## Notes
- The dataset used is **synthetic** and intended for learning and portfolio demonstration.
- Screenshots are included to preview dashboard insights without opening Excel.
- This dashboard demonstrates **end-to-end Excel analytics**, from cleaned data to executive reporting.
