# Healthcare Claims & Cost Analysis Dashboard

## Project Overview
This project analyzes healthcare claims data to understand patient volume, encounter patterns, payer coverage, and uncovered healthcare costs. The objective is to provide clear, decision-ready insights using Excel.

## Business Problem
Healthcare organizations need visibility into:
- How many patients are being served
- What types of encounters occur most frequently
- Which payers contribute the most coverage
- How much cost remains uncovered and poses financial risk

This dashboard helps answer these questions in a simple and interactive way.

## Dataset
Synthetic healthcare dataset containing information on patients, encounters, procedures, and payers.

## Tools Used
- Microsoft Excel
- PivotTables
- KPIs
- Charts & Slicers

## Key KPIs
- Total Unique Patients
- Total Encounters
- Total Healthcare Cost
- Total Payer Coverage
- Total Uncovered Amount

## Key Insights
- Ambulatory encounters account for the highest volume of visits.
- Medicare and Medicaid are the top contributors to payer coverage.
- The highest number of encounters occurred in 2014.
- Boston patients account for the highest total healthcare claim amount.

## Deliverables
- Interactive Excel dashboard
- KPI-driven executive summary

## How to Use This Project
1. Download the Excel file from the `dashboard` folder
2. Open `PROJECT.xlsx`
3. Use slicers to filter by City and Encounter Type
4. Review KPIs and charts for insights
